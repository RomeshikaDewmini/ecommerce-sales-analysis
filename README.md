# E-Commerce Sales Analysis Dashboard

## 📌 Project Overview
An interactive Business Intelligence dashboard analyzing e-commerce sales data to uncover revenue opportunities, customer behavior patterns, and operational efficiency insights.

## 🛠️ Tools Used
- **Power BI** - Interactive Dashboard & Visualizations
- **MySQL** - Data Analysis & Queries
- **Excel** - Initial Data Exploration & Cleaning

## 📊 Dashboard Highlights
- **Executive Summary:** KPIs for Total Sales, Profit, Orders, and Profit Margin
- **Product Analysis:** Top 10 Products, Category-wise performance, Sub-Category matrix
- **Regional Analysis:** Interactive map with region and state performance
- **Time Analysis:** Monthly trends, Year-over-Year comparison
- **Customer Insights:** Top customers, Segment analysis

## 🔍 Key Insights
1. Technology category contributes **38%** of total sales
2. Western region generates **35%** of total revenue
3. Sales peak during **November and December**
4. Top 10 customers contribute **15%** of total revenue
5. Year-over-Year growth rate: **12%**

## 💡 Business Recommendations
- Increase inventory for top-selling products
- Focus marketing on high-performing regions
- Implement loyalty program for top customers
- Optimize promotional activities for Q4 season

1. Download the dataset from Kaggle
2. Import CSV to MySQL using the provided schema
3. Open Power BI file and refresh data connection
4. Explore interactive dashboard
